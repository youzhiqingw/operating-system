## 课程学习建议

- 适当提前预习，可以寒暑假阶段先学着
- 理解背诵的点比较多，有些也会比较抽象
- 在学校上课可以当温习再巩固一遍
- 每学一个单元，结合分享给大家的习题复习巩固
- 一些重点知识点要掌握理解，面试、考研须知！
- 牛客等一些平台也有相关操作系统的题目，大家如果练完了分享资料里的，可以去那练

## 操作系统实验

- 认真实操，重点是学Linux操作，Linux知识
- 可以结合专业知识进行实验课学习，搭建服务器环境

## 复习备考

- 教学平台的练习题过关熟记掌握——选择题原题！
- 复习PPT上的练习习题要背诵熟记——大题原题！
